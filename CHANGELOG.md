# Changelog

## v1.0.0

### Features
- Git worktree display with smart branch deduplication
- Last commit message with relative age indicator
- Changed files count with emoji indicator
- Test failure indicator
- Jira ticket extraction from branch names with OSC 8 hyperlinks
- GitHub PR status with CI check results and commit count
- Git stash warning indicator
- Localhost dev server detection with clickable ports
- Active MCP server display
- Full ANSI 256 color theming
- OSC 8 hyperlink support for modern terminals
